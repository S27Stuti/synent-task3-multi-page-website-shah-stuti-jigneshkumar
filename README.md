## 📖 Project Overview

This project is a frontend multi-page website designed to demonstrate web development skills using core web technologies. It includes multiple interconnected pages, responsive layouts, and interactive elements created with JavaScript.

The website is optimized for different screen sizes to provide a seamless user experience on desktops, tablets, and mobile devices.
